# Glossary

* AP: Active Pool
* BO: Borrower Operations
* CCR: Critical Collateral Ratio (for Recovery mode)
* DP: Default Pool
* MCR: Minimum Collateral Ratio (usually set to 110%)
* SP: Stability Pool
